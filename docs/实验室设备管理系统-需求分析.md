### 需求1：用户注册

**需求描述**：新用户注册，默认分配学生角色

**实现方式**：

- 前端收集用户名、密码、真实姓名、手机号
- 后端校验用户名唯一性，查询STUDENT角色ID，创建用户并设置status=1

**核心代码**：

```java
// AuthService.java - 注册逻辑
public void register(RegisterRequest request) {
    // 检查用户名是否已存在
    Long count = userMapper.selectCount(new LambdaQueryWrapper<User>()
            .eq(User::getUsername, request.getUsername()));
    if (count != null && count > 0) {
        throw new BusinessException("用户名已存在");
    }

    // 查询学生角色作为默认角色
    Role studentRole = roleMapper.selectOne(new LambdaQueryWrapper<Role>()
            .eq(Role::getRoleCode, "STUDENT"));

    // 创建新用户，默认启用
    User user = new User();
    user.setUsername(request.getUsername());
    user.setPassword(request.getPassword());
    user.setRealName(request.getRealName());
    user.setPhone(request.getPhone());
    user.setRoleId(studentRole.getId());
    user.setStatus(1);
    userMapper.insert(user);
}
```

### 需求2：用户登录

**需求描述**：用户名+密码登录，Session会话管理

**实现方式**：

- 前端发送用户名密码到后端
- 后端校验用户存在性和密码正确性
- 登录成功后将用户信息存入Session

**核心代码**：

```java
// LoginController.java - 登录逻辑
@PostMapping("/login")
public Result login(@RequestBody User user, HttpSession session) {
    User loginUser = userService.lambdaQuery()
            .eq(User::getUsername, user.getUsername())
            .one();

    if (loginUser == null) {
        return Result.error("账号不存在");
    }

    if (!loginUser.getPassword().equals(user.getPassword())) {
        return Result.error("密码错误");
    }

    session.setAttribute("loginUser", loginUser);
    return Result.success(loginUser);
}
```

### 需求3：权限拦截

**需求描述**：未登录用户无法访问受保护接口

**实现方式**：

- 实现HandlerInterceptor拦截器
- 检查Session中是否有loginUser
- 无登录信息则返回401未登录提示

**核心代码**：

```java
// LoginInterceptor.java - 权限拦截器
@Component
public class LoginInterceptor implements HandlerInterceptor {

    @Override
    public boolean preHandle(HttpServletRequest request,
                             HttpServletResponse response,
                             Object handler) throws Exception {
        Object loginUser = request.getSession().getAttribute("loginUser");

        if (loginUser == null) {
            response.setContentType("application/json;charset=utf-8");
            response.getWriter().write("{\"code\":500,\"msg\":\"用户未登录，请先登录\"}");
            return false;
        }
        return true;
    }
}
```

```java
// WebConfig.java - 注册拦截器
@Configuration
public class WebConfig implements WebMvcConfigurer {
    @Override
    public void addInterceptors(InterceptorRegistry registry) {
        registry.addInterceptor(loginInterceptor)
                .addPathPatterns("/api/**")
                .excludePathPatterns("/api/auth/login", "/api/auth/register");
    }
}
```

// UserController.java - 用户管理接口
@RestController
@RequestMapping("/api/users")
@RequireRole({"ADMIN"})
public class UserController {
    
    @GetMapping
    public Result<Page<User>> page(@RequestParam(defaultValue = "1") long page,
                                   @RequestParam(defaultValue = "10") long size,
                                   @RequestParam(required = false) String keyword) {
        LambdaQueryWrapper<User> wrapper = new LambdaQueryWrapper<>();
        if (keyword != null && !keyword.isBlank()) {
            wrapper.like(User::getUsername, keyword)
                    .or().like(User::getRealName, keyword);
        }
        return Result.ok(userService.page(new Page<>(page, size), wrapper));
    }

    @PostMapping
    public Result<Void> add(@RequestBody User user) {
        userService.save(user);
        return Result.ok();
    }

    @PutMapping("/{id}")
    public Result<Void> update(@PathVariable Long id, @RequestBody User user) {
        user.setId(id);
        userService.updateById(user);
        return Result.ok();
    }

    @DeleteMapping("/{id}")
    public Result<Void> delete(@PathVariable Long id) {
        userService.removeById(id);
        return Result.ok();
    }
}

### 需求6：设备查询

**需求描述**：分页查询，支持按名称/编号/分类/状态筛选

**实现方式**：

- 使用MyBatis-Plus的LambdaQueryWrapper构建动态查询条件
- 支持keyword模糊匹配设备名、编号、位置

**核心代码**：

```java
// DeviceController.java - 设备查询
@GetMapping
public Result<Page<Device>> page(@RequestParam(defaultValue = "1") long page,
                                 @RequestParam(defaultValue = "10") long size,
                                 @RequestParam(required = false) String keyword,
                                 @RequestParam(required = false) Long categoryId,
                                 @RequestParam(required = false) Integer status) {
    LambdaQueryWrapper<Device> wrapper = new LambdaQueryWrapper<>();
    if (keyword != null && !keyword.isBlank()) {
        wrapper.and(w -> w.like(Device::getDeviceName, keyword)
                .or().like(Device::getDeviceNo, keyword)
                .or().like(Device::getLocation, keyword));
    }
    if (categoryId != null) {
        wrapper.eq(Device::getCategoryId, categoryId);
    }
    if (status != null) {
        wrapper.eq(Device::getStatus, status);
    }
    wrapper.orderByDesc(Device::getCreateTime);
    return Result.ok(deviceService.page(new Page<>(page, size), wrapper));
}
```

### 需求12：审批申请

**需求描述**：管理员/实验员审批，通过/拒绝并填写备注

**实现方式**：

- 审批通过(status=1)时生成借用记录并更新设备状态
- 审批拒绝(status=2)时仅更新申请状态

**核心代码**：

```java
// BorrowBusinessServiceImpl.java - 审批
@Override
@Transactional
public void approve(ApproveBorrowRequest request) {
    BorrowApply apply = borrowApplyMapper.selectById(request.getApplyId());
    if (apply.getStatus() != 0) {
        throw new BusinessException("该申请已经审批过");
    }

    apply.setStatus(request.getStatus());
    apply.setApproveUserId(AuthContext.getUserId());
    apply.setApproveTime(LocalDateTime.now());
    borrowApplyMapper.updateById(apply);

    if (request.getStatus() == 1) {  // 通过
        // 生成借用记录
        BorrowRecord record = new BorrowRecord();
        record.setApplyId(apply.getId());
        record.setDeviceId(apply.getDeviceId());
        record.setUserId(apply.getUserId());
        record.setBorrowTime(LocalDateTime.now());
        record.setStatus(1);
        borrowRecordMapper.insert(record);

        // 更新设备状态为已借出
        Device device = deviceMapper.selectById(apply.getDeviceId());
        device.setStatus(2);
        deviceMapper.updateById(device);
    }
}
```

### 需求18：处理报修

**需求描述**：实验员更新维修状态、填写维修结果

**实现方式**：

- 前端打开处理模态框，选择状态、填写结果

**核心代码**：

```java
// RepairBusinessServiceImpl.java - 处理报修
@Override
@Transactional
public void handle(RepairHandleRequest request) {
    RepairRecord record = repairRecordMapper.selectById(request.getRepairId());
    record.setRepairStatus(request.getRepairStatus());
    record.setRepairResult(request.getRepairResult());
    if (request.getRepairStatus() == 2) {  // 维修完成
        record.setFinishTime(LocalDateTime.now());
    }
    repairRecordMapper.updateById(record);

    if (request.getRepairStatus() == 2) {
        Device device = deviceMapper.selectById(record.getDeviceId());
        device.setStatus(1);  // 恢复为可借
        deviceMapper.updateById(device);
    }
}
```
